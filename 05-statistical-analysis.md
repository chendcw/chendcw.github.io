## Core Methodologies

### 1. Hypothesis Testing
| Test | Application | SAS Code | R Code |
|------|-------------|----------|--------|
| t-test | Mean comparison | `PROC TTEST` | `t.test()` |
| ANOVA | Multi-group means | `PROC GLM` | `aov()` |
| Chi-square | Categorical data | `PROC FREQ` | `chisq.test()` |
